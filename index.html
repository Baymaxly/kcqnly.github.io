JAVA TCP编程

TCP：Transmission Control Protocol

  • 传输控制协议，面向连接的协议
  • 两台机器之间可靠的，无差错的数据传输
  • 双向字节流传递
  1. 服务器（软件服务器：能实现一定功能，在公开地址上对外服务）
    创建一个ServerSocket，等待连接
  2. 客户机：创建一个Socket，连接到服务器
  3. 服务器:ServerSocket接收到连接，创建一个Socket和客户的Socket创建专线连接，后续服务器和客户机的对话（这一对Socket）会在一个单独的线程（服务器端）上运行
  4. 服务器继续等待连接，返回步骤1
